# Football Manager Game


## About

Welcome to My Football Team Management Game. The aim in this game is to be the first in the league at the end of 38 weeks. Each team has a team strength and a team strategy that you can always change.
Team Powers: Team strength is selected between 0 and 100. You can change it from the settings section before the game starts.

## Team Strategies Features

Aggressive: It is the most offensive strategy in the game. It is a game strategy based on playing the team forward and increasing the goal of increasing the chance of scoring by putting pressure on the opponent.
Semi-Agesif: Although it has an aggressive game strategy, it is a softened version
Normal: It is a balanced strategy in the game. The probability of scoring and conceding a goal is based on team strength
Semi-Defensive: Although it has a defensive game strategy, it is a softened version
Defensive: It is the most defensive game strategy in the game and the least probability of conceding a goal.

## How to Open File
You can play the game by running the main.py file directly. You can view the matches and scoreboard by progressing each week.
